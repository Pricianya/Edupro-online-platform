# 📚 EduPro Analytics Dashboard

## Instructor Performance and Course Quality Evaluation

### 📖 Overview
EduPro Analytics Dashboard is an interactive data analytics project developed using Python and Streamlit. The dashboard analyzes instructor performance, course quality, and learner engagement to help educational platforms make data-driven decisions. It provides visual insights into teaching effectiveness, course ratings, experience, and expertise-based performance.

---

## 🚀 Features

- 📊 Interactive KPI Dashboard
- 👨‍🏫 Instructor Performance Leaderboard
- 🏆 Top & Bottom Performing Instructors
- 📈 Experience vs Teacher Rating Analysis
- 📚 Course Category & Course Level Analysis
- 💼 Expertise-wise Performance Comparison
- 🔥 Course Quality Heatmap
- 📉 Correlation Analysis
- 📅 Enrollment Trend Analysis
- 🔍 Dynamic Filters
- 📥 Download Filtered Data
- 📋 Executive Summary
- 💡 Business Recommendations

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn

---

## 📂 Dataset

The project uses four datasets:

- Teachers
- Courses
- Transactions
- Users

The data is integrated using **TeacherID** and **CourseID** to perform comprehensive analysis.

---

## 📊 Key Performance Indicators (KPIs)

- Average Teacher Rating
- Average Course Rating
- Rating Consistency Index
- Experience Impact Score
- Enrollment Influence Ratio
- Total Instructors
- Total Courses
- Total Enrollments

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/EduPro-Analytics-Dashboard.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
